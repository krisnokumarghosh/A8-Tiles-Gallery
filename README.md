# Tiles Gallery

A modern tile shop web application where users can explore, search, and view detailed information about premium tile collections.

---

## Live URL

[Visit Tiles Gallery](https://a8-tiles-gallery.vercel.app)

---

## Purpose

Tiles Gallery is a full-featured e-commerce style showcase platform for a tile shop. It allows customers to browse featured and full tile collections, search by name, and view detailed product information — all behind a secure authentication system.

---

## Key Features

- **Live Marquee Updates** — The home page displays a scrolling marquee showcasing the latest tile arrivals and updates in real time.

- **Full Gallery** — Browse all available tiles on the Gallery page in a clean, organized layout.

- **Search & Filter** — Filter tiles instantly by title using the search field on the Gallery page.

- **Tile Detail Page** — Click the "See Details" button on any tile card to view full product details. Redirects to login if the user is not authenticated.

- **Authentication System**
  - Login with email and password
  - Register with name, email, password, and photo URL
  - Google Sign-In and Sign-Up support

- **Profile Management** — Logged-in users can edit their profile, including updating their name and photo URL.

- **Mobile Swipe** — Tile cards on small devices support swipe gestures via Swiper.js for a smooth mobile experience.
